Commands by category

f Files & Find

Telescope pickers & saving — files on disk: see Oil

SpcffFind files
SpcfgLive grep — also Spc /
SpcfbOpen buffers
SpcfrResume last picker
SpcfhHelp tags
SpcfpSwitch project
SpcfsSave file · Spcfsa save all

- Files on disk (Oil)

A directory as an editable buffer — no sidebar tree

-Open parent directory — again to keep climbing
EnterOpen file · enter directory
q · CtrlcClose, restore previous buffer
:wApply edits to disk — asks to confirm
dd · pDelete · move (cut here, paste in another dir)
g?Show every Oil key
g.Toggle hidden files (on by default here)
CtrlpPreview the entry under the cursor

w Windows

Splits & navigation · numbers in the winbar

Spcwh/j/k/lFocus left / down / up / right
Spcw/Split right
Spcw-Split down
SpcwdClose window
Spcw=Balance windows — equal share again
SpcwH/J/K/LMove window far left / down / up / right
Spc1…9Jump to window (winbar number)

b Buffers

The bufferline tabs across the top

SpcbnNext buffer
SpcbpPrevious buffer
SpcbdClose buffer

S Sessions

persistence.nvim · auto-resumes on bare nvim

SpcSfSwitch project (Telescope picker)
SpcSsRestore this dir's session
SpcSlRestore last session
SpcSdDon't save this session

g h Git

Neogit (equal-width split) · lazygit (TUI) · gitsigns (hunks)

SpcgsOpen status (Neogit)
SpcggGit TUI, floating (lazygit)
]c / [cNext / previous hunk
SpchsStage hunk
SpchrReset hunk
SpchpPreview hunk
SpchbBlame line
s c p FIn Neogit: stage · commit · push · pull

KHover docs
SpcrnRename symbol

SpcxxProject diagnostics
SpcxXCurrent-buffer diagnostics

j Jump & move

flash.nvim (avy) · treesitter code motions

SpcjjFlash jump — label & teleport
SpcjtFlash treesitter — select a scope
]m / [mNext / prev function
]] / [[Next / prev class
]M · ][…end variants of the above

a i Text objects

Prefix with v/c/d — e.g. vaf, cif

af / ifa / inner function
ac / ica / inner class
aa / iaa / inner parameter (argument)
Spca / SpcASwap parameter next / prev

g y Editing

Comment · surround · pairs

gccToggle line comment · gc range · gbc block
ys{m}{c}Surround: add (S in visual)
cs{o}{n}Surround: change
ds{c}Surround: delete
Tab / ⇧TabTab out of / back into a pair
jfExit insert mode (→ Esc)

' Terminal

Right-side split, equal width, opens in view mode

Spc'Toggle terminal — far right, equal share
i · jfStart typing · back to view mode
Spccsvisual — send selection to terminal

m Markdown

render-markdown.nvim — starts raw, toggle to render

SpcmToggle rendered view — per buffer
rendersHeadings, code blocks, bullets, checkboxes, tables, callouts
cursor lineDe-renders to raw markup so you can edit in place

z Folds

Treesitter-driven — follow real code structure

zc · zo · zaClose · open · toggle fold
zR / zMOpen all / close all folds
